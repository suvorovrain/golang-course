# InfoRepo CLI

Простая консольная утилита на **GoL**, которая позволяет быстро получить информацию о любом публичном репозитории GitHub через REST API.

## Возможности
Программа выводит следующие данные:
* Имя репозитория
* Описание
* Количество звёзд
* Количество форков
* Дату создания

## Установка и компиляция

Перед использованием утилиту необходимо скомпилировать. Убедитесь, что у вас установлен [Go](https://go.dev).

Скомпилируйте программу:
   ```bash
   go build inforepo.go
   ```

## Использование

### Справка
Для получения списка всех доступных флагов используйте команду:
```bash
./inforepo --help
```

### Параметры
* `--owner` — имя владельца репозитория (пользователя или организации) на GitHub.
* `--repo` — название конкретного репозитория.

### Пример запуска
Чтобы узнать информацию о боте **UniversityManagerBot**, выполните:
```bash
./inforepo --owner AndyBarnv --repo UniversityManagerBot
```
