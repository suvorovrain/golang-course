# GitHub Repo CLI Tool

Простой CLI-инструмент на Go для получения информации о репозитории GitHub.

## Функциональность

Инструмент выводит следующую информацию о репозитории:

- Имя репозитория
- Описание
- Количество звёзд
- Количество форков
- Дата создания


## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone git@github.com:mshipilov5/golang-course.git
cd golang-course
```

2.	Запустите инструмент:

```bash
go run main.go -owner=<имя владельца> -repo=<название репозитория>
```

3. Вывод:
```
   Repository: linux  
   Description: Linux kernel source tree  
   Stars: 220601  
   Forks: 60734  
   Created at: 2011-09-04 22:48:12 +0000 UTC  
```

## Обработка ошибок
-   Если не переданы данные о репозитории - выводится подсказка, как вводить данные  
-	Если репозиторий не найден — выводится сообщение Repository not found.  
-	При сетевых ошибках или неверном вводе — выводится соответствующая ошибка.
